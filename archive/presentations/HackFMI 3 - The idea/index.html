Софтуер за управление на залите в образователните институции*

*или поне какво ще се опитаме да направим

Кои сме ние?

  • Група първокурсници, завършили ТУЕС
  • Занимаваме се предимно с програмиране вместо да учим математики
  • Общите ни проекти са предимно web
  • 2 на 2 сме относно C# и Java
  • Винаги пишем скоби на if с тяло от един ред!
  • И не на последно място - `Парчетата`

Проблемът

Какво не е наред?

  • Липса на информация за заетостта на залите
  • Трудност при намиране на свободна зала
  • Бавно и трудно изграждане на учебна програма

Ние имаме желание да поправим това

Основни цели

  • REST API с пълна информация за всичко
  • SPA за визуализация и възможност за запазване на стая
  • Модул за изграждане на програма

И за това нещо имаме 3 дни

В крайна сметка да направим Open Source приложение, което помага на всички!

Благодарим за вниманието
